# Hexplore

A Linux terminal hex editor using only ANSI C and escape sequences. This program is the long awaited text user interface for my other program chastehex.

However, this is not a replacement for chastehex. Rather it serves a different purpose. It is meant for exploring the hexadecimal of a file you are not familiar with and see what it contains. You can modify the bytes and see the effect on the file by then opening it in other programs, including chastehex to see if your changes were saved correctly.

But the best part is that it runs in a terminal and displays help information at the bottom so you know how to operate it. I got this idea from the nano text editor and how easy it is to use. There is nothing to memorize and it makes this the most convenient hex editor for viewing or editing small files.

There is currently no Windows version of this program, but I may eventually consider learning enough Windows API to make one. It is not a high priority though because I despise Windows and don't use it unless required for a school class.
